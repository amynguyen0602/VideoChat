Chatterbox is a video chat application that allows users to join Video Room and interact with other users.
Basic functions:
- Signin/ Signup
- Update User Info
- Create contact/ Delete contact
- Create new room/ Edit room/ Delete room
- Join Room to chat with other users

Tech stacks:
- Front-end: ReactJS
- Back-end: NodeJS and ExpressJS
- Database: MongoDB
- External API: Twillio

Instruction: 
- Unzip the file
- Go to server folder using command /* cd server */
- Install dependencies in server folder using command /* npm install */
- Go to client folder using command /* cd client */
- Install dependencies in client folder using command /* npm install */
- Go back to server folder using command /* cd .. */
- Run the application using /* npm run dev */

Application URL: https://videochatterbox.herokuapp.com
